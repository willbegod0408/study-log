 # 📚 Python 기초 정리

## 자료형 기호
| 자료형 | 기호 |
|--------|------|

## 리스트 복사 방법
```python
b = a[:]       # 슬라이싱 복사
b = copy(a)    # copy 모듈

```

## 비교 연산자
| 연산자 | 설명 |
|--------|------|
| `x < y` | x가 y보다 작다 |
| `x > y` | x가 y보다 크다 |
| `x == y` | x와 y가 같다 |
| `x != y` | x와 y가 같지 않다 |
| `x >= y` | x가 y보다 크거나 같다 |
| `x <= y` | x가 y보다 작거나 같다 |

## 조건문 (if/elif/else)
- 조건문 다음 콜론(`:`) 필수
- `else`는 `if` 없이 독립 사용 불가
- 조건이 참이면 if 블록, 거짓이면 else 블록 수행

---

## match-case 문
- `match` 뒤에 비교할 변수
- `case` 뒤에 비교할 값
- `case _` : 그 외 모든 값 (와일드카드, else 역할)
- `"A" | "B" | "C"` : 여러 값 중 하나와 일치하면 수행

---

## 조건부 표현식
```python
변수 = 참일_때_값 if 조건 else 거짓일_때_값
```

# 📚 Python 03장 - 제어문 정리

## if 문

### 기본 구조
```python
if 조건문:
    실행할_문장
else:
    실행할_문장
```
- 조건문 뒤 콜론(`:`) 필수
- 들여쓰기로 블록 구분
- `else`는 `if` 없이 단독 사용 불가

### and / or / not
- `and` : 둘 다 참이면 참
- `or` : 하나라도 참이면 참
- `not` : 참/거짓 반전
- **실무 팁**: `not`은 헷갈리니까 남발하지 말고 조건을 뒤집어 쓰는 게 명확함

### in / not in
| 표현 | 의미 |
|------|------|
| `x in 리스트` | x가 리스트에 있으면 참 |
| `x not in 리스트` | x가 리스트에 없으면 참 |

### 한 줄 if
```python
if 'money' in pocket: print("택시")
```
- 실행 코드가 1개일 때만 가능
- 2개 이상이면 들여쓰기 방식 필수

### 조건부 표현식
```python
변수 = 참일_때_값 if 조건 else 거짓일_때_값
```

---

## while 문

### 기본 구조
```python
변수 = 초기값
while 조건문:
    실행할_문장
```
- 조건이 참인 동안 반복
- 거짓이 되는 순간 탈출
- **변수 초기값 선언 필수** (안 하면 NameError)

### != 연산자
- `==` : 같다
- `!=` : 같지 않다

### break / continue
- `break` : while문 즉시 탈출
- `continue` : 조건문 처음으로 돌아가기

### 무한 루프
```python
while True:
    ...
```
- 탈출: `Ctrl + C`

---

## for 문

### 기본 구조
```python
for 변수 in 리스트(또는 튜플, 문자열):
    실행할_문장
```
- 리스트의 값을 하나씩 꺼내서 변수에 담음
- 변수명은 자유 (i, num, x 뭐든 OK)

### range() 함수
```python
range(1, 11)  # 1부터 10까지 (끝 숫자는 미포함)
```

### print의 end 옵션
```python
print(i, end=" ")  # 줄바꿈 대신 공백
print('')          # 줄바꿈 역할
```

### 리스트 컴프리헨션
```python
result = [num * 3 for num in a if num % 2 == 0]
```
- 리스트 + for문 + if문을 한 줄로 압축

### zip() 함수
```python
for name, kor, eng in zip(names, korean, english):
    print(f"{name}: 국어 {kor}점, 영어 {eng}점")
```
- 여러 리스트를 동시에 묶어서 반복
