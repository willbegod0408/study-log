  정수형 말 그대로 정수 소수점없는 수 와 0 실수형 소수점이 포함된 수 하지만 1과 1.0은 다른 자료 값은 같지만  아주큰수 아주작은수는 4.24e 혹은 4.24E로 표현 
8진수(octal)는 0o 또는 0O(숫자 0 + 알파벳 소문자 o 또는 대문자 O)로 시작한다. a += 1은 a = a + 1과 동일하다. 이처럼 +=를 복합 연산자라 부른다.
파이썬은 0부터 숫자를 셈 하지만 뒤에서 부터세는 음수일때는 0부터 세지않음 문자열 슬라이싱 [0:4]이면 4번째숫자는 세지않음 3번째까지만 끝번호는 수식x 공백문자도 일반적인 문자와 동일하게 취급해서 ‘life’ 와 ‘life ‘는 다름 문자열 포매팅에서 %d %s d는 숫자 s는 문자를 사용한다 2개이상값을 넣을때는 %다음 괄호를 사용해서 컴마를 이용해서 각각의 값을 넣어 주면 된다 
%s	문자열(String)
%c	문자 1개(character)
%d	정수(Integer)
%f	부동소수(floating-point)
%o	8진수
%x	16진수
%%	Literal % (문자 % 자체)
하지만 %s는 모든값을 문자열로 바꾸어 대입 무적임
%의 경우 포매팅연산자로 쓰이고 또 안에서 문자열로 쓰일경우 문자열로 쓰일경우에는 포매팅연사자가 있고 문자열로 쓰일경우에는 반드시 %%2개로 사용한다
%10s는 전체 길이가 10개인 문자열 공간에서 대입되는 값을 오른쪽으로 정렬하고 그 앞의 나머지는 공백으로 남겨 두라는 의미다.
오른쪽 정렬은 :< 대신 :>을 사용하면 된다. 화살표의 방향을 생각하면 어느 쪽으로 정렬되는지 바로 알 수 있다.:^를 사용하면 가운데 정렬도 가능하다.
